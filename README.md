# Gastos Compartidos

Aplicación estática para visualizar gastos por ciclo, categorías y cuotas.

## Publicación en GitHub Pages

1. Sube este repositorio a GitHub.
2. Asegúrate de que la rama principal se llame `main`.
3. En GitHub, entra a Settings > Pages y selecciona GitHub Actions como fuente.
4. El workflow en [.github/workflows/deploy-pages.yml](.github/workflows/deploy-pages.yml) publicará el sitio automáticamente.

La app usa un archivo HTML, CSS y la carpeta de imágenes, por lo que funciona como sitio estático.
